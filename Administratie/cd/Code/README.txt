
Alle code op deze cd is geschreven door Olivier Van den Eede in 2017.
Eventuele wijzigingen en updates zijn te vinden op github: https://github.com/4ilo/Bachelorproef_E-ICT

- Ets-import is een laravel(php) project dat gebruikt wordt om een ETS5 XML configuratiefile in te laden in het KNX-IoT platform.

- OliKNXPlatform is een HomeBridge (https://github.com/nfarina/homebridge) plugin voor de koppeling met KNX-IoT.

- Sim-KNX is een C++ programma dat via UART communiceerd met een Sim-KNX module van Tapko.

- De webconfig is een browser configuratie-tool voor het aanpassen van de JSON config files.

- De webpage is een busturing van het KNX-IoT platform via de browser.
